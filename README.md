# Prácticas para el curso Diseño con lógica programable

## Introducción
En este curso se exploró la implementación de varios sistemas utilizando Verilog HDL y varios programas de simulación, para programar varias prácticas en una tarjeta FPGA. Además, se exploraron varios conceptos dentro de la lógica programable relevantes a la carrera Ingenieria en Robótica y Sistemas Digitales.
